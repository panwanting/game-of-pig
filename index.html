Start Game of Pig

1 v.s 1 dices game

Game Rule

遊戲目標

玩家輪流骰骰子，當總分大於100即獲勝

遊戲規則:

當輪流骰骰子，當骰子數字為2~6，可以加總分數

玩家可以隨時選擇是否停止，將此回合骰到分數加總

當玩家骰到數字1，此回合骰到的分數都不算，並直接輪到下一位玩家
